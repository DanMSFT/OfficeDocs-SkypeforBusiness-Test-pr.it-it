# Cmdlet per VoIP aziendale

 

_**Ultima modifica dell'argomento:** 2014-03-19_

VoIP aziendale è l'implementazione Microsoft di Voice over IP (VoIP). I cmdlet disponibili per la gestione di VoIP aziendale in Microsoft Lync Server 2013 consentono di creare e modificare dial plan (precedentemente noti come profili località), criteri vocali, route e regole di normalizzazione.

## Cmdlet per VoIP aziendale

La maggior parte delle attività di gestione relative a VoIP aziendale può essere eseguita dal Pannello di controllo di Lync Server. Le stesse attività possono essere eseguite con i cmdlet da Lync Server Management Shell o da uno script, per automatizzare particolari attività. Di seguito è riportato un elenco di cmdlet direttamente correlati alla gestione di VoIP aziendale:
