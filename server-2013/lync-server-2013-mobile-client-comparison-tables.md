# Tabelle di confronto dei client mobili per Lync Server 2013

 

_**Ultima modifica dell'argomento:** 2015-07-17_

Nelle tabelle vengono confrontate caratteristiche e funzionalità tra client mobili Lync 2013 e il client desktop Lync 2013 nelle seguenti categorie:

  - Accesso, notifiche Push e caratteristiche generali

  - Presenza avanzata

  - Contatti e gruppi di contatto

  - Messaggistica istantanea

  - Audio e video da Lync a Lync

  - Servizi di conferenza

  - Telefonia

  - Utenti esterni

  - Archiviazione e conformità

Queste tabelle indicano le funzionalità disponibili agli utenti di Lync in una distribuzione di Lync Server 2013 locale. Le stesse funzionalità sono disponibili anche agli utenti di Skype for Business online e Microsoft Office 365 se non diversamente specificato nelle note a piè pagina delle tabelle.


> [!NOTE]
> <UL>
> <LI>
> <P>Per la Guida e per risorse online per gli utenti finali, visitare "Microsoft Lync 2013 per client mobili" all'indirizzo <A href="http://go.microsoft.com/fwlink/?linkid=286237">http://go.microsoft.com/fwlink/?LinkId=286237</A>.</P>
> <LI>
> <P>Per confrontare le caratteristiche disponibili in altri client di Lync 2013, vedere <A href="lync-server-2013-desktop-client-comparison-tables.md">Tabelle di confronto dei client per Lync Server 2013</A>.</P>
> <LI>
> <P>Lync Server 2013 supporta anche le app per dispositivi mobili di Lync 2010. Per informazioni dettagliate, vedere "Tabelle di confronto dei client mobili" nella documentazione di Lync Server 2010 all'indirizzo <A class=uri href="http://go.microsoft.com/fwlink/?linkid=234777">http://go.microsoft.com/fwlink/?linkid=234777</A>.</P></LI></UL>
