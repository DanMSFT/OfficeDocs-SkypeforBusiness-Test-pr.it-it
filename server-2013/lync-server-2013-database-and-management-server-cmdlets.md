# Cmdlet per server di gestione e database

 

_**Ultima modifica dell'argomento:** 2012-10-09_

I cmdlet per server di gestione e database vengono utilizzati per gestire sia i database back-end di Microsoft Lync Server 2013 che i servizi di gestione front-end. È possibile utilizzare questi cmdlet per installare o disinstallare qualsiasi database utilizzato da Lync Server 2013, nonché per configurare il punto di controllo del servizio Active Directory per l'archivio di gestione centrale.
