# Cmdlet per telefoni e dispositivi

 

_**Ultima modifica dell'argomento:** 2012-06-28_

In Microsoft Lync Server 2013 sono disponibili diversi cmdlet che consentono di gestire telefoni e altri dispositivi hardware, inclusi telefoni VoIP (Voice over IP), telefoni di area comune, ad esempio un telefono situato nell'ingresso di un edificio, in una mensa o in un altro luogo pubblico, e persino telefoni analogici, ovvero telefoni non in grado di eseguire Lync Phone Edition.

## Cmdlet per telefoni e dispositivi

È possibile utilizzare i cmdlet **CsDeviceUpdate** per gestire il servizio Web di aggiornamento dispositivi, un componente di Lync Server che consente agli amministratori di distribuire aggiornamenti firmware a telefoni e ad altri dispositivi che eseguono Lync Phone Edition.
