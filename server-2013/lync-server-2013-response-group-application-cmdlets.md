# Cmdlet per l'applicazione Response Group

 

_**Ultima modifica dell'argomento:** 2012-06-26_

applicazione Response Group è una funzionalità di gestione delle chiamate di VoIP aziendale che instrada le chiamate in arrivo verso gruppi di agenti.
