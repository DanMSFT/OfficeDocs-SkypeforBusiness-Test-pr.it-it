# Cmdlet per l'archiviazione e il monitoraggio

 

_**Ultima modifica dell'argomento:** 2012-10-04_

I cmdlet per l'archiviazione e il monitoraggio consentono agli amministratori di gestire l'archiviazione delle sessioni di messaggistica istantanea e di conferenza, di registrare informazioni dettagliate sulle chiamate e di monitorare Microsoft Lync Server 2013 mediante la qualità percepita dagli utenti (QoE).


> [!NOTE]
> Per ulteriori informazioni sui cmdlet, vedere il blog di Windows PowerShell per Lync Server all'indirizzo <A class=uri href="http://go.microsoft.com/fwlink/?linkid=263432%26clcid=0x410">http://go.microsoft.com/fwlink/?linkid=263432&amp;clcid=0x410</A>. Il contenuto di ogni blog e i relativi URL sono soggetti a modifiche senza preavviso.
