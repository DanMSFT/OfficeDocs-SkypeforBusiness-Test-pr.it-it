# Cmdlet del server chat persistente

 

_**Ultima modifica dell'argomento:** 2012-06-27_

I cmdlet per Chat persistente consentono di gestire e configurare il servizio Chat persistente di Microsoft Lync Server 2013, noto in precedenza come servizio Group Chat. Il servizio Chat persistente consente agli utenti di partecipare a sessioni di messaggistica istantanea online. Sebbene queste sessioni possano avvenire in tempo reale, il contenuto di ogni sessione è persistente e ciò significa avere la possibilità di riprendere la conversazione in qualsiasi momento.
