# Cmdlet per il servizio Enhanced 9-1-1

 

_**Ultima modifica dell'argomento:** 2012-06-20_

Microsoft Lync Server 2013 viene fornito con cmdlet che consentono di implementare e gestire l'implementazione del servizio chiamate di emergenza (Enhanced 9-1-1) di una soluzione VoIP aziendale. Utilizzare questi cmdlet per eseguire il mapping tra punti di connessione e indirizzi fisici e per configurare le impostazioni necessarie affinché gli utenti di VoIP aziendale possano eseguire chiamate di emergenza e inviare automaticamente una posizione al provider dei servizi di emergenza. Non è possibile configurare il servizio chiamate di emergenza (E9-1-1) dal Pannello di controllo di Lync Server. A tale scopo, è necessario utilizzare i cmdlet.
