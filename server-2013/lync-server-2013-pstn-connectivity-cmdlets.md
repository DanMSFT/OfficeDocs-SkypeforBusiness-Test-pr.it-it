# Cmdlet per la connettività PSTN

 

_**Ultima modifica dell'argomento:** 2012-10-09_

VoIP aziendale offre impostazioni che abilitano chiamate verso e dalla rete PSTN (Public Switched Telephone Network) senza alcuna riduzione della qualità del servizio (QoS). È possibile configurare queste impostazioni mediante i cmdlet disponibili in Lync Server Management Shell.
