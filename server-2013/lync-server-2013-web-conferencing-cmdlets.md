# Cmdlet per le conferenze Web

 

_**Ultima modifica dell'argomento:** 2013-07-29_

Le conferenze e le riunioni online sono elementi importanti in Microsoft Lync Server 2013. I cmdlet CsConferencingConfiguration e CsConferencingPolicy sono gli strumenti di amministrazione principali per la gestione delle conferenze tramite Windows PowerShell.

## Cmdlet per le conferenze Web
