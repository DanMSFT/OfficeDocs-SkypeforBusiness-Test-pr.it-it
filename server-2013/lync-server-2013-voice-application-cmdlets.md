# Cmdlet per le applicazioni vocali

 

_**Ultima modifica dell'argomento:** 2012-10-09_

Le applicazioni vocali vengono eseguite nel servizio applicazioni nel Front End Server. Tali applicazioni facilitano la gestione delle chiamate telefoniche in arrivo e in uscita, consentendo di parcheggiare le chiamate, gestire i numeri non assegnati, coordinarsi con la Messaggistica unificata di Exchange ed eseguire l'applicazione Response Group.
