# Cmdlet per il servizio Controllo di ammissione di chiamata

 

_**Ultima modifica dell'argomento:** 2012-03-21_

Il servizio Controllo di ammissione di chiamata determina se la larghezza di banda della rete è sufficiente per stabilire una sessione audio o video in tempo reale di qualità accettabile. Controllo di ammissione di chiamata viene gestito configurando limiti e impostando reti, siti e subnet, nonché le relative interazioni.
